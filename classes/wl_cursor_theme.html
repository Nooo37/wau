Class wl_cursor_theme

A wl_cursor_theme is a set of wl_cursors with a unified appearance.

Thus the normal workflow is to load a cursor theme through load and to then retrieve the needed wl_cursor through get_cursor. Once you got a wl_cursor, you can access its wl_cursor_images through the wl_cursor.image property. Then you can get the images buffer with the wl_cursor_image.get_buffer method. After that you can attach the buffer to a surface and run set_cursor on the surface whose cursor you want to set.

You can look up on how it can be done in the xdg_top.lua example.

See also:

    xdg_top.lua

Usage:

    local ctheme = wau.wl_cursor_theme.load(nil, 24, shm)
local cursor = ctheme:get_cursor("cross")
